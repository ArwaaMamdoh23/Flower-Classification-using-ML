# Flower Classification using Machine Learning

## Overview
Machine learning project for classifying flower images using computer vision techniques.

## Objective
Build a model that can correctly classify different flower types from images.

## Dataset
Labeled flower images. Preprocessed using resizing, normalization, and train/test split.
source : https://www.kaggle.com/datasets/marquis03/flower-classification

## Methodology
- Load dataset  
- Preprocess images  
- Encode labels  
- Split data  
- Train model (CNN or ML classifier)  
- Evaluate performance  
- Make predictions  

## Model
- Convolution layers for feature extraction  
- Pooling layers for reduction  
- Fully connected layers for classification  
- Softmax output  

## Evaluation
- Accuracy  
- Loss

- NumPy
